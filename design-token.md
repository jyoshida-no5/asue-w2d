# Design Token
> 参考サイト: https://www.yakiniku-king.jp/
> 生成日: 2026-04-15
> 適用対象: samples/asue/index.html（お好み焼きはここやねん）
> ユーザー方針: 「写真が映えるようにシンプルに。モバイルUIUXを意識」

---

## デザインコンセプト

焼肉きんぐの参考サイトから読み取った「空気感」：
- **写真ファースト**: 大判ビジュアルが主役。UI要素は写真を引き立てる脇役に徹する
- **ダーク×ゴールド×ホワイト**: 深みのある暗色背景に、金・赤アクセント、白の清潔感を組み合わせる
- **余白の贅沢さ**: セクション間を広めに取り、各写真・コンテンツが「呼吸している」感覚
- **力強いタイポグラフィ**: 太く、追跡性の高いゴシック体で食欲・信頼感を演出
- **シンプルなナビ**: 「予約/店舗検索」を最重要CTAとして際立たせる

## タイポグラフィ

フォント構成:
- 見出し: `Noto Serif JP` (font-weight: 700) — 格式・食の誇り
- 本文: `Noto Sans JP` (font-weight: 400/600) — 可読性・清潔感

| 要素 | Tailwindクラス |
|---|---|
| キャッチコピー (h1) | `font-serif font-bold text-4xl sm:text-5xl leading-tight tracking-tight` |
| セクション見出し (h2) | `font-serif font-bold text-2xl sm:text-3xl leading-snug` |
| カード見出し (h3) | `font-bold text-base sm:text-lg leading-snug` |
| 英語ラベル（SECTION ID） | `text-xs font-bold tracking-[0.2em] uppercase` |
| 本文 | `text-sm leading-relaxed` |
| キャプション・価格 | `text-sm font-bold` |
| 極小テキスト | `text-xs leading-relaxed` |

## 余白

| 区分 | Tailwindクラス | 備考 |
|---|---|---|
| セクション間（縦） | `py-16 lg:py-24` | ゆったりとした呼吸感 |

| カード間ギャップ | `gap-4 sm:gap-6` | |

## レイアウト

- **グリッド基本構造**: 
  - ヒーロー: 全幅画像 + 下からのグラデーションオーバーレイ + テキスト
  - おすすめカード: SP = 横スクロール（75vw幅カード）、PC = 3カラム
  - こだわり: SP = 2カラムグリッド、PC = 3カラム
  - 店舗CTA: センタリング + 検索バー
  - ニュース: リスト形式
  - クーポンCTA: 全幅アクセントカラー帯

- **非対称・重なり**: なし（シンプル重視・写真を邪魔しない）
- **全幅要素の使い方**: ヒーロー・クーポンCTAは画面幅100%使用。コンテンツセクションは `max-w` でコントロール

---

## 装飾・テクスチャ

- **影**: カードに `shadow-sm`（控えめ）。写真はシャドウなし
- **ボーダー**: `border border-[#e5e0d8]` の細線のみ。角丸は `rounded-2xl`（カード）、`rounded-full`（ボタン）
- **背景テクスチャ**: 単色のみ（グラデーションは使わない。ただしヒーローオーバーレイは `bg-gradient-to-t from-black/70 via-black/20 to-transparent`）
- **ホバーエフェクト**: `transition-colors duration-200` で色変化のみ
- **写真**: 全てグレープレースホルダー `bg-zinc-300`（`img-placeholder`クラス付き）

## セクション構造パターン

### ヘッダー
- 固定（`fixed top-0`）、高さ `h-14` (SP) / `h-16` (PC)
- 背景: `#1a1008`（漆黒）
- ロゴ: ゴールドテキスト + セリフ体
- SPナビ: ハンバーガー（右端）
- PCナビ: テキストリンク（右寄り）+ 「店舗を探す」ゴールドボタン

### ヒーロー
- 全幅、高さ `h-[480px] lg:h-[600px]`
- グレープレースホルダー
- 下部グラデーションオーバーレイ
- テキスト: 白（h1 大きめ）+ サブコピー小さめ + CTAボタン2つ（ゴールド + 白/透明）

### コンテンツセクション（おすすめ等）
- 英語サブラベル（RECOMMEND等）→ 日本語見出し → コンテンツ → CTAボタンの順
- 背景交互: ホワイト / クリーム / ダーク

### カード
- 白背景、rounded-2xl、shadow-sm
- 上部: アスペクト比 16:9 画像プレースホルダー
- 下部: カテゴリラベル（小）→ タイトル → 説明 → 価格

### フッター
- 背景: `#1a1008`（漆黒）
- ロゴ + ナビリンクグリッド（PC: 4カラム）
- コピーライト行: ボーダー上

### SPボトムナビ
- `sticky bottom-0`、高さ `h-[60px]`
- 白背景、`border-t`
- 4アイテム: 電話 / 店舗検索（ゴールドBG） / LINE（グリーンBG） / クーポン

## モバイルUIUX 強化指針（ユーザー要求）

1. **タップターゲット最小48px**: ボトムナビアイテム・CTAボタン全てh-12以上
2. **フォントサイズ最小14px**: 本文は`text-sm`以上
3. **横スクロールカード**: おすすめセクションは `snap-x mandatory` で自然なスクロール
4. **ボトムナビ固定**: コンバージョン最重要3アクション常時表示（電話、店舗、LINE）
5. **ヒーローCTA**: SP用は縦積みではなく、画面下部にボタンが来るよう配置調整
6. **セーフエリア対応**: `pb-[env(safe-area-inset-bottom)]` 考慮
